# Cybersecurity Basics – Introduction

## Description

This directory contains basic Bash scripts created as part of the **Cybersecurity Basics** module.
Each script focuses on a fundamental Linux or cybersecurity concept such as system information, password generation, hashing, permissions, and process management.

## Files

* **0-release.sh**
  Displays the Linux distribution ID in a concise single-line output.

* **1-gen_password.sh**
  Generates a random password.

* **2-sha256_validator.sh**
  Validates a SHA256 hash against a given input.

* **3-gen_key.sh**
  Generates a cryptographic key.

* **4-root_process.sh**
  Displays processes running as root.

## Usage

Make scripts executable:

```bash
chmod +x *.sh
```

Run a script:

```bash
./0-release.sh
```

## Requirements

* Bash shell
* Linux environment
* Scripts must be executable
* Output must follow project constraints
